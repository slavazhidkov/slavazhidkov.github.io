Windcatcher

Search engine for the best places for wind turbines around the world. Visit site.

October, 2018

Overview

I was responsible for visual design (web and mobile) and user experience. It’s basically a visual data base for the best places for wind turbines around the world. The user can find the best potential location and if he wants to get an access to the detailed map and specifics of this area - he can purchase an access.
